## Why Use Tags?
Tags help you quickly categorize and filter notes across notebooks.

## How to Add Tags
<Steps>
  <Step title="Edit a note">
    ````
    Open any note and click the Tags field at the top.
    ````
  </Step>
  <Step title="Add or create tags">
    ````
    Type to select existing tags or create new ones by pressing Enter.
    Multiple tags can be added per note.
    ````
  </Step>
  <Step title="Search by tags">
    ````
    Use the search bar or filter menu to view all notes with a tag.
    ````
  </Step>
</Steps>
