## Recent Updates

Stay informed about the latest releases. You can check your current version in the app settings under `Account > Version`.

<Update label="2024-10-15" description="v0.3.0" tags={["feature", "improvement"]}>

- Added **Report Builder** with drag-and-drop fields, sorting, grouping, and filters for custom task reports
- Introduced **Customise Card Layout** allowing you to toggle fields, reorder, and adjust hover interactions

- Enhanced **Create a Mixed View** for combining task lists dynamically

- Updated login flow with "Keep me logged in for 7 days" option and better session management
- Improved search with clear results counter and "Clear search" button
- Added timezone loading indicator for accurate task scheduling

## Bug Fixes

- Fixed scratch pad counter display issues
- Resolved unsaved changes confirmation dialogs for better UX

</Update>

## Migration Guide

Most updates are seamless, but review these steps for v0.3.0 if you encounter issues.

<Steps>
  <Step title="Backup Your Data" icon="database">

  Export your tasks via the Report Builder before updating.

  ```
  1. Open Report Builder
  2. Select all fields
  3. Generate and download CSV
  ```

  </Step>
  <Step title="Update the App" icon="download">

  Refresh your browser or clear cache.

  <CodeGroup tabs="Browser,Progressive Web App">
  ```bash
  # Hard refresh
  Ctrl+Shift+R (Windows/Linux)
  Cmd+Shift+R (macOS)
  ```
  ```bash
  # PWA: Check for updates in app menu
  # Or reinstall from minddump.uk
  ```
  </CodeGroup>

  </Step>
  <Step title="Verify Changes" icon="check-circle">

  Test new features:

  - Customize card layout
  - Build a sample report
  - Confirm session persists

  </Step>
</Steps>

<Callout kind="alert" title="Breaking Changes">
In v0.3.0, menu expansion now defaults to `300ms` delay. Adjust in settings if instant hover is preferred. No data loss occurs.
</Callout>
